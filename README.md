# Web-Project
Projekti ne Web1
Event & Hobby Platform

Ky projekt është një aplikacion web për menaxhimin e eventeve dhe komuniteteve hobby, i ndërtuar për të demonstruar përdorimin praktik të HTML, CSS, JavaScript dhe jQuery në një aplikacion front-end funksional.

Platforma u mundëson përdoruesve të:

zbulojnë evente lokale,

krijojnë evente të reja,

shohin eventet që kanë krijuar vetë,

eksplorojnë dhe bashkohen në grupe hobby sipas interesave.

Projekti funksionon pa backend dhe përdor localStorage për ruajtjen e të dhënave të eventeve të krijuara nga përdoruesi.

Karakteristikat Kryesore

Krijim dhe ruajtje e eventeve përmes forme interaktive

Shfaqje automatike e eventeve personale në faqen “Eventet e Mia”

Kërkim i eventeve lokale sipas qytetit

Menaxhim i grupeve hobby me pamje detajesh dhe galeri imazhesh

Dizajn modern, i animuar dhe plotësisht responsive

Teknologjitë e Përdorura

HTML5 – Strukturimi i përmbajtjes

CSS3 – Dizajn, animacione dhe responsive layout

JavaScript (ES6) – Logjika e aplikacionit

jQuery – Manipulimi i DOM dhe event handling

localStorage – Ruajtja e të dhënave në browser

Arkitektura e Projektit

Projekti është i organizuar në faqe të ndara sipas funksionalitetit:

Events – shfaqja dhe filtrimi i eventeve lokale

Create Event – krijimi dhe ruajtja e eventeve të reja

My Events – shfaqja e eventeve të krijuara nga përdoruesi

Hobby Groups – eksplorimi dhe bashkimi në grupe komunitare

Të gjitha faqet ndajnë të njëjtin stil vizual dhe navigim të përbashkët.

Ekzekutimi i Projektit

Projekti nuk kërkon server apo databazë.
Mjafton të hapet cilado faqe .html në një browser modern.

Qëllimi i Projektit

Qëllimi kryesor është demonstrimi i:

strukturimit të një aplikacioni web front-end,

ruajtjes së të dhënave në browser,

ndarjes së logjikës nga dizajni,

dhe ndërtimit të një UI/UX funksionale dhe intuitive.
